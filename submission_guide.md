# HIRO 2026 投稿指南 / Submission Guide

> 投稿邮箱：**hirolab_hiro2026@163.com**

---

## 投稿要求 / Submission Requirements

请在投稿邮件中**务必包含**以下信息，否则可能影响录用：

Please ensure your submission email includes **all** of the following:

### 1. 投稿类别 / Submission Category

请从以下类别中选择一项，并在邮件主题或正文中明确标注：

Please select one of the following categories and clearly indicate it in your email subject or body:

| 类别 / Category | 说明 / Description |
|---|---|
| **Research Paper / Abstract** | 研究论文、摘要、学术性探讨 |
| **Fan Art / Illustration** | 同人图、插画、摄影、视觉艺术作品 |
| **Fan Fiction / Creative Writing** | 同人文、短篇小说、诗歌 |
| **Other Creative Work** | 其他创意形式（音乐、视频、互动作品等） |

### 2. 作品信息 / Work Information

- **作品标题 / Title**
- **作者署名（可使用昵称）/ Author name (pen name accepted)**
- **作品简介（1-2 句）/ Brief description of the work (1–2 sentences)**

### 3. 展示形式 / Presentation Format（可多选）

| 形式 / Format | 说明 |
|---|---|
| ☐ **口头汇报 / Oral Presentation** | 8–10 分钟现场报告，汇报者将获得注册费全额退还（现场报销） |
| ☐ **展板展示 / Poster Display** | 组委会统一印刷 A4 展板，投稿人自行美工设计 |
| ☐ **仅投稿论文集 / Proceedings Only** | 仅收录电子版论文集 |

> 口头汇报与展板展示均可同时选择论文集收录。不选择口头或展板者，也可仅收录论文集。

### 4. 联系方式 / Contact Information

请在邮件中提供**至少一种**常用联系方式，以便组委会发送录用通知及相关信息：

Please provide **at least one** of the following contact methods for receiving acceptance notifications:

- **电子邮箱 / Email**（推荐 / recommended）
- **QQ 号**
- **微信号 / WeChat ID**
- **其他即时通讯方式 / Other IM**

### 5. 作品附件 / Attachment

- 论文/摘要：请以 **PDF** 格式提交
- 同人图/插画（用于论文集收录）：请以 **PNG/JPG（300 dpi）** 格式提交
- 同人图/插画（如选择展板展示）：请以 **A4 大小、300 dpi** 的 PNG/PDF 格式提交
- 同人文/创意写作：请以 **PDF 或纯文本** 格式提交
- 其他格式：请附上可预览的文件或链接
